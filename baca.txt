Commit Message:
    Format:
        git commit -m "tipe: message"

    Contoh:
        git commit -m "feat: sistem login"

    Tipe:
        chore    : Commit yang berkaitan dengan konfigurasi seperti .gitignore
        feat     : Commit yang berkaitan dengan fitur baru
        fix      : Commit yang berkaitan dengan perbaikan bug
        docs     : Commit yang berkaitan dengan dokumentasi
        refactor : Commit yang berkaitan dengan perubahan nama seperti variabel, function, file dan directory
        style    : Commit yang berkaitan dengan perubahan desain
